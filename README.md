## Testing with testcontainers-go

Follow the steps below to run tests using `testcontainers-go`:

1. Clone the repository:
```bash
$ git clone https://github.com/y16ra/testcontainers-go-demo.git
```
2. Run the tests:
```bash
$ cd testcontainers-go-demo
$ make test
```
